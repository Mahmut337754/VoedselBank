# Voedselbank Dag 2 - Software Development

## 📌 Project

Dit project is ontwikkeld voor het examen Software Development (EP1 - Dag 2).
De applicatie ondersteunt een voedselbank bij het beheren van klanten, producten, leveranciers en voedselpakketten.

## ⚙️ Functionaliteiten

* CRUD voor klanten, producten, leveranciers en voedselpakketten
* Dashboard met navigatie
* Leveranciersoverzicht toegankelijk via de dashboardkaart "Leveranciers"
* Authenticatie (login/registratie)
* Database met relaties (MySQL)
* MVC-structuur toegepast

## 🧱 Technieken

* PHP (MVC)
* MySQL
* Git & GitHub

## 🚀 Installatie

1. Clone de repository
2. Importeer de database (`VoedselbankSql_dag2.sql`)
3. Configureer database in `config/database.php`
4. Start de applicatie via localhost
